Référentiel de levures

Le besoin en azote de chaque levure est utilisé dans les variables calculées.

Variables calculées

Tapez une formule commençant par =. Variables disponibles : TAP, AzoteAssimilable, Vitesse, Temperature, Densite, DeltaInit, BesoinAzoteLevure (0=faible, 1=moyen, 2=élevé), ainsi que le nom de toute autre variable déjà créée.
Exemple : =TAP*2,5 - AzoteAssimilable + BesoinAzoteLevure*30

Alertes globales

Ces alertes s'appliquent à tous les comptes.
Dans le message : {NomVariable} insère la valeur calculée avec son unité, {NomVariable:unite} l'unité seule.
Exemple : ajouter {BesoinAzoteTotal} à la cuve → « ajouter 45,30 mg/L à la cuve »
Réactivable coché = l'alerte se redéclenche après correction. Décoché = une fois validée, elle ne se redéclenche plus pour cette cuve.
